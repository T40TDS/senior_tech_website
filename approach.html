Goal

In order to achieve inertial sensor fusion. I made use of RFID technology. The eventual goal is to achieve 6-DOF approximation.

Obstacles

A number of obstacles are faced in accurate pose estimation. Error can result from a number of sources the sensor itself can pose internal issues, the primary and most mathematical source of error is integration error. I aimed to correct this error with RFID technology.
